# Lumen Release Notes

This directory contains release notes for all versions of Lumen, including alpha and pre-release versions.

- [v0.0.1-alpha-20251229](./v0.0.1-alpha-20251229.md) - 2025-12-29

  - Fixed gitignore configuration
  - Improved build reliability

- [v0.0.1-alpha-20251228](./v0.0.1-alpha-20251228.md) - 2025-12-28
  - Initial public alpha release
  - Core Kubernetes management features
  - Real-time monitoring and multi-cluster support

## Versioning Scheme

Lumen follows semantic versioning with the following format:
- `v0.0.1-alpha-YYYYMMDD` - Alpha releases (daily builds during alpha phase)
- `v0.0.1-beta.X` - Beta releases (feature-complete, testing phase)
- `v0.1.0` - First stable release
